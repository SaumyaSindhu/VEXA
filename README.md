## 🌟 Overview

This project is a **full-stack AI chat application** inspired by platforms like Perplexity and ChatGPT. It combines **modern frontend architecture**, **scalable backend design**, and **agentic GenAI systems** to deliver a powerful conversational experience.

Whether you're evaluating engineering depth or exploring GenAI systems — this project showcases **real-world production patterns**.

---

## 🧠 Core Highlights

### 🔥 Under the Hood (Engineering Depth)

* **Multi-LLM Architecture**

  * Gemini → Conversational + streaming responses
  * Mistral → Smart chat title generation
* **Agentic AI (Tool Calling)**

  * Dynamically decides when to:

    * 🌐 Search internet (real-time data)
    * 📄 Query user PDFs (RAG)
    * 🧠 Answer directly
* **Streaming AI (SSE)**

  * Token-by-token responses (ChatGPT-like)
  * Smooth typing UX with buffered rendering
* **RAG (Retrieval-Augmented Generation)**

  * Upload PDFs → chunk → embed → store → retrieve
  * Context-aware answers from user documents
* **Context-Aware AI**

  * Maintains full conversation history
  * Uses message role mapping (`user`, `ai`)
* **Scalable State Management**

  * Normalized Redux store (O(1) access)
* **Secure Authentication**

  * JWT + HTTP-only cookies
* **Backend Architecture**

  * MVC + service layer + middleware separation
* **Rate Limiting**

  * Global + AI-specific protection (prevents abuse)

## 💬 Chat System Architecture

```text
User Message
   ↓
Saved in DB
   ↓
Context fetched
   ↓
Agent (LLM decides)
   ├── Direct answer
   ├── Internet search
   └── RAG (PDF retrieval)
   ↓
Streaming response (SSE)
   ↓
Saved in DB → UI updated
```

---

## ⚡ Streaming Architecture

```text
LLM → token stream → SSE → frontend reader → Redux update → UI typing effect
```

✔ Smooth

## 📄 RAG Pipeline
